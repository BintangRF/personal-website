What I’m good at?

HTML & CSS

Saya memiliki pemahaman dasar tentang HTML dan CSS yang cukup untuk membuat website sederhana, mengatur tata letak konten, mengatur warna dan gaya teks, serta menambahkan elemen visual yang menarik.

Bootstrap & TailwindCSS

Saya juga memiliki pengalaman menggunakan Bootstrap dan Tailwind CSS dalam pengembangan web. Dengan Bootstrap, saya dapat dengan mudah membuat tata letak responsif dan menambahkan komponen UI yang modern dan estetis. Sementara itu, dengan Tailwind CSS, saya dapat mengoptimalkan styling dengan pendekatan utility-first, mempercepat proses pengembangan dan membuat kode lebih mudah dipelajari dan dipelihara.

JavaScript

Saya memiliki pengalaman dalam menggunakan JavaScript untuk memanipulasi DOM, seperti menambahkan atau mengubah elemen HTML, mengatur atribut, dan menangani peristiwa seperti klik atau submit form. Dengan JavaScript DOM, saya dapat membuat interaksi dinamis dalam halaman web, seperti menampilkan pesan pop-up, mengubah warna teks, atau menganimasikan elemen.

ReactJs

Saya juga telah menggunakan ReactJS untuk mengembangkan aplikasi web Frontend yang dinamis dan responsif. Dengan ReactJS, saya dapat membuat komponen UI yang terpisah dan reusable, mengelola state aplikasi dengan efisien menggunakan hooks seperti useState dan useEffect, serta membuat navigasi antar halaman dengan React Router. Dengan pendekatan komponen dan virtual DOM yang dimiliki ReactJS, pengembangan aplikasi web menjadi lebih terstruktur dan mudah di-maintain.
